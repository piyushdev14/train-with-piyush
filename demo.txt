shortcut keyt
win + ctrl + d - for new virtual desktop
win + ctrl + left/right - for switching left and right on virtual desktop in windows.
win + ctrl + f4  - for closing current tab
win + i = to open setting.
win + e = to open file explorer
win + m = to minimize all window 
win + shift + m = to restore minimize window
win + d = to show desktop/again to restore work
